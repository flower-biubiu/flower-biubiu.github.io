322. 零钱兑换

给你一个整数数组 coins ，表示不同面额的硬币；以及一个整数 amount ，表示总金额。

计算并返回可以凑成总金额所需的 最少的硬币个数 。如果没有任何一种硬币组合能组成总金额，返回 -1 。

你可以认为每种硬币的数量是无限的。

总结：
最优子结构(可以画出一棵树)+填表

一维数组的含义：
横坐标: 总金额
DP[i]: 总金额为i时，最少需要的硬币数

最优子结构：
当前总金额最少硬币数 由 小于且加和为当前总金额的数目的最小硬币数 加1 的最小值构成

  • 初始化状态（边界状态）DP[0] = 0
  • 状态参数（变量）amount coins
  • 状态转移方程（递推公式）F(i) = min(F(i - coins[j]) + 1)

备用思路（可能超时）：

  • 贪心+回溯
  • 暴力递归
  • 备忘录递归（记忆化搜索）

5. 最长回文子串

Tips: 下面两种方式初始化数组的区别在于，fill会用同一个空数组引用来初始化整个数组

let arr = new Array(s.length).fill([])
for(let i = 0; i < s.length - 1; i++) {
    arr.push([])
}

300. 最长递增子序列

给你一个整数数组 nums ，找到其中最长严格递增子序列的长度。

子序列 是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。例如，[3,6,2,7] 是数组 [0,3,1,6,2,2,7] 的
子序列
。

最优子结构：

  • 以第i个元素结尾的子序列的最大长度 等于 小于i的结尾的最大长度加一
  • 以第i个元素结尾的子序列的最大长度 的最大值
